Trail Feature Test

Fading Trails Implementation

This test demonstrates the new fading trails feature for both Termites and Langton's Ants. The trails fade over time and can be configured with different lengths and opacity levels.

  • Trail Length: Number of steps before the trail completely fades (default: 30)
  • Trail Opacity: Maximum opacity of the trail (0.1 to 1.0)
  • Trail Enabled: Toggle trails on/off
  • Dynamic Colours: Trails use the same dynamic colour scheme as the actors
